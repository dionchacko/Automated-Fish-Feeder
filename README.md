# Automated Fish Feeder 🐟

## 📌 Project Description
This project is an automated fish feeding system using ESP32, RTC module, and a motor mechanism. It feeds fish at scheduled times without manual intervention.

## ⚙️ Components Used
- ESP32
- RTC Module
- Servo Motor / DC Motor
- Fish Feed Container
- Power Supply

## 🔌 Working
The RTC keeps track of time. At predefined feeding times, the ESP32 activates the motor to release food.
